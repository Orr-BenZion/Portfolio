My object and I

This project is a followup to a task that required each of us to bring an object of significance to class.

The object I chose was a 'Stormtrooper' figurine (a character from the StarWars series).
In this project we had to create a self portrait in which we combine the elements from the Object in the previous assignment.

In my Portrayal, a descending line splits the portrait into two parts, me on one side, slightly cartoonified, and the Stormtrooper on the other side, we may look separate but we're two sides of the same coin, together we are whole.
